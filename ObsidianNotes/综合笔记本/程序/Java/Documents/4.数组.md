# 4.数组

## 声明数组变量

```java
int[] myList;
double[] myList;
```

## 创建数组

```java
myList = new double[10];
// 声明和创建可以一起写
double[] myList = new double[10];
```

## foreach循环

```java
for (double e: myList){
    System.out.println(e);
}
```

## 复制数组

```java
System.arraycopy(sourceArray, srcPos, targetArray, tarPos, length);
```

## Arrays类

```java
java.util.Arrays
```

## 二维数组

```java
double[][] myMatrix;
```
